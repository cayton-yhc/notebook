# XOR题型

## XOR基本运算法则
*   1、交换律：a^b=b^a
*   2、结合律：a^(b^c)=(a^b)^c
*   3、自消律：a^a=0
*   4、和0异或不变：a^0=a

## 怎么靠？
*   1、结合配合交换，在公式推导时常用于消去或移动项
*   2、自消，当题目指明“偶数”时，往往需要通过移项两两消去；“奇数”，则留下一个
*   3、特殊用法，异或换值，不需要额外创建变量——
        a=a^b
        b=a^b
        a=a^b

        我来推导一下——设a=A,b=B
        a=a^b -> a=A^B
        b=a^b -> b=(A^B)^B=A
        a=a^b -> b=(A^B)^A=B
